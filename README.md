# Supplementary Information

This repository contains supplementary materials for the submitted manuscript "XX" from the Monash Computational Chemistry Group.

The files contained in this repository are: 

- QM7b-wB97X-D-AEs.csv: Reference atomisation energies of QM7b calculated at the wB97X-D/aug-cc-pVQZ level.
- AE-QM7b-xyz.zip: An archive of labelled QM7b structures referenced by QM7b-wB97X-D-AEs.csv.
- Reference.yaml: A .yaml file containing all reference energies, alongside commands used to calculate the reference energies, and constituent systems.
- PredictionsModel1.csv: Raw data of predicted and reference energies of the machine learned model after being trained on atomisation and interaction energies.
- PredictionsModel2.csv: Raw data of predicted and reference energies of the machine learned models after being retrained with reaction energies and reaction barriers.
- TAE_DFT_Comparison.csv: TAE140 atomisation energies as calculated by common DFT functionals.
- EINT_DFT_Comparison.csv: Interaction energies as calculated by common DFT functionals.
